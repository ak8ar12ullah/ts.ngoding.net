## Typescript harus di compile ke Javascript

Ketika kita menggunakan file name TS maka file tersebut tidak bisa digunakan oleh
sistem kita. karena TypeScript diciptakan bukan untuk menggantikan Javascript tapi menambahkan Fitur ke Javascript.

Dikarenakan hal tersebut, ketika kita ingin mengguankan TypeScript maka kita harus melakukan build untuk mengubahnya menjadi script javascript.

## Instalasi Typescript

Untuk melakukan build tersebut maka kita harus menginstal TypeScript ke Computer kita.

Instalasi ini guna digunakan untuk proses build kita.

Pertama-tama kita harus sudah menginstall NodeJS ke Computer kita.

Setelah NodeJS terinstal, Maka di NodeJS tersebut sudah ada NPM maupun PNPM.

maka kita harus menginstallnya menggunakan perintah di bawah pada CMD di windows ataupun Terminal di Linux pada tempat project berada.

import { Tabs, TabItem } from "@astrojs/starlight/components";

<Tabs>
  <TabItem label="NPM">``` npm install typescript --save-dev ```</TabItem>
  <TabItem label="PNPM">``` pnpm add typescript --save-dev ```</TabItem>
</Tabs>

atau kalau kita ingin typescript dapat digunaakn dimana saja secara global maka kita tinggal menjalankan script dibawah.

<Tabs>
  <TabItem label="NPM">``` npm install typescript -g ```</TabItem>
  <TabItem label="PNPM">``` pnpm add typescript -g ```</TabItem>
</Tabs>
